Method etag

Generates an Etag

Parameters:

  • url must be a String.
    (URL requested)

  • size must be a Number.
    (Response size)

  • modified must be a Number.
    (Modified time)

  • body must be an Object.
    ([Optional] Response body)

Returns a String
(Etag value)

TurtleIO.prototype.etag = function ( /*url, size, modified, body*/ ) { return this.hash( array.cast( arguments ).join( "-" ) ); };
