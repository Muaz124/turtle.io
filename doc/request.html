Request handler which provides RESTful CRUD operations

Parameters:

    (HTTP(S) request Object)

  • res must be an Object.
    (HTTP(S) response Object)

  • host must be a String.
    ([Optional] Virtual host)

if ( req.headers.expect ) { return this.error( req, res, this.codes.EXPECTATION_FAILED ); }

Determining if the request is valid
