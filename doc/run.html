Method run

Runs middleware in a chain

Parameters:

  • req must be an Object.
    (Request Object)

  • res must be an Object.
    (Response Object)

  • host must be a String.
    ([Optional] Host)

  • method must be a String.
    (HTTP method)

Returns an Object
(TurtleIO instance)
