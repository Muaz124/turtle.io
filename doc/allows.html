Method allows

Determines which verbs are allowed against a URL

Parameters:

  • uri must be a String.
    (URI to query)

  • host must be a String.
    (Hostname)

  • override must be a Boolean.
    (Overrides cached version)

Returns a String
(Allowed methods)

TurtleIO.prototype.allows = function ( uri, host, override ) { var self = this, timer = precise().start(), verbs = ["delete", "get", "post", "put", "patch"], result; result = verbs.filter( function ( i ) { return self.allowed( i, uri, host, override ); } ); result = result.join( ", " ).toUpperCase().replace( "GET", "GET, HEAD, OPTIONS" ); timer.stop(); this.dtp.fire( "allows", function () { return [host, uri, timer.diff()]; } ); return result; };
